### Steps to run normal chat
 1. Run Ollama in cmd ollama run 
 ```
 deepseek-r1:1.5b
 ```

 2. Open DEEPSEEK-OLLAMA-LOCAL terminal
    2.1 Check if myenv is active
        if not actived run ```  .\myenv\Scripts\activate  ```
3. Run command  ``` streamlit run app.py ```
